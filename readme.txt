== Description ==

This plugin lets you change the default cart page notices for WooCommerce.This includes the added to cart, removed from cart, and cart is empty notices.  

== Installation ==

1. Download the .zip file
2. Upload and extract the contents of the zip file to your wp-content/plugins/folder
3. Activate the plugin from your WP-admin / Plugins
4. Enjoy!

== Frequently Asked Questions ==

= What does this plugin do? =

This plugin will let you change the default cart page notices for WooCommerce to your own custom notices. This includes the "added to cart", "removed from cart", and "cart is empty" notices.

= Where can I find the settings for this plugin? =

The settings are found in your "WooCommerce > Settings > Products" settings page.
